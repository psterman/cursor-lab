# 赛博朋克主题颜色修复说明

## 修复内容

将右侧抽屉中所有不符合赛博朋克主题的颜色统一为绿色系（`#00ff41` / `var(--accent-terminal)`）。

## 修复的颜色列表

### 1. 蓝色系 → 绿色
- `text-blue-400` → `text-[var(--accent-terminal)]`
- `bg-blue-600` → `bg-[var(--accent-terminal)]` （合并按钮）
- `#93c5fd` (浅蓝) → `#00ff41`
- `rgba(59, 130, 246, ...)` (蓝色渐变) → `rgba(0, 255, 65, ...)`

### 2. 靛蓝/紫色系 → 绿色
- `text-indigo-200` → `text-[var(--accent-terminal)]`
- `border-indigo-400/40` → `border-[var(--accent-terminal)]/40`
- `bg-indigo-500/5` → `bg-[var(--accent-terminal)]/5`
- `#a855f7` (紫色) → `rgba(0,255,65,0.7)`
- `purple-500/20` → `[var(--accent-terminal)]/20`

### 3. 青绿色系 → 统一绿色
- `text-emerald-400` → `text-[var(--accent-terminal)]`
- `bg-emerald-400` → `bg-[var(--accent-terminal)]`

### 4. 其他绿色 → 统一使用 CSS 变量
- `text-green-500` → `text-[var(--accent-terminal)]`
- `text-green-400` → `text-[var(--accent-terminal)]`
- `text-green-400/90` → `text-[var(--accent-terminal)]/90`

### 5. 黄色警告色 → 绿色（为统一主题）
- `text-yellow-400` → `text-[var(--accent-terminal)]`
- `border-yellow-500` → `border-[var(--accent-terminal)]`
- `text-yellow-300` → `text-[var(--accent-terminal)]`

## 保留的颜色

以下颜色保留，用于特定用途：

### 红色系（错误/警告）
- `text-red-500` - LEVEL_CRITICAL 标签
- `border-red-500/30` - 错误边框
- `bg-red-950/20` - 错误背景
- `text-red-300/400` - 错误文字
- `#f87171` - 排名下降指示

### 蓝色（校准模式指示）
- `#3b82f6` - 校准中状态的光标颜色（与正常绿色区分）

## 主题色彩规范

### 主色调
```css
--accent-terminal: #00ff41;  /* 赛博朋克绿 */
```

### 使用场景
- **主要文字**: `text-[var(--accent-terminal)]`
- **背景高亮**: `bg-[var(--accent-terminal)]`
- **边框**: `border-[var(--accent-terminal)]`
- **半透明**: `text-[var(--accent-terminal)]/90`, `bg-[var(--accent-terminal)]/20`

### 辅助色
- **背景**: `#050505`, `rgb(24 24 27)` (zinc-900)
- **文字**: `#ffffff`, `rgb(161 161 170)` (zinc-400)
- **边框**: `rgba(255,255,255,0.1)`, `#27272a`
- **错误**: `#f87171` (red-400)

## 验证

运行以下命令检查是否还有非主题颜色：
```bash
# 检查蓝色、紫色、粉色、橙色
grep -n "text-blue\|text-indigo\|text-purple\|text-pink\|text-orange\|text-cyan\|text-emerald" stats2.html

# 检查十六进制颜色（排除绿色和灰度）
grep -n "#[0-9a-fA-F]\{3,6\}" stats2.html | grep -v "#00ff41\|#050505\|#27272a\|#ffffff\|#f87171"
```

## 效果

修复后，右侧抽屉的所有元素都使用统一的赛博朋克绿色主题，视觉效果更加一致。
