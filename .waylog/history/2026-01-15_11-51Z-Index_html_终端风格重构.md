**User**

目标：将当前的 index.html 样式进行全面重构。去掉过度装饰的紫色渐变和圆润感，转而采用极简、高对比度、带有 “黑客 / 终端监控” 质感的程序员审美风格。

具体修改要求：
色彩体系收敛（Terminal Palette）：

背景：保持纯黑 #050505，删除所有 radial-gradient 的彩色柔光背景。

核心主色：将 --accent-purple 替换为经典的终端绿（如 #00ff41 或 #10b981）或冷调高亮蓝（如 #3b82f6）。

文字颜色：主文字使用纯白 #ffffff，次要文字使用中灰 #71717a。

边框：使用极细的深灰边框 #27272a。

字体与排版（Digital Layout）：

全站等宽：强制 body 使用 'JetBrains Mono', monospace，确保所有文字（包括标题）都具备代码感。

去渐变文字：取消 h1 和标题的渐变填充，改用实色，并增加 letter-spacing: 0.1em 和 text-transform: uppercase（全大写）。

终端装饰：在所有主要标题前自动添加 > 符号或伪元素模拟终端提示符（如 > SYSTEM_REPORT）。

组件重塑（Component Hardening）：

去圆角化：将所有卡片（.stat-card, .chart-card, .vibe-badge）的 border-radius 从 20px 统一降至 2px 或 4px（直角感）。

背景处理：卡片背景取消模糊效果，改为纯黑或微透的深灰 rgba(24, 24, 27, 0.8)。

增加扫描线感：为 body 增加一个极细微的 CSS 网格背景或点阵背景（Dot Matrix），模拟旧式监视器。

交互与细节（Geek Details）：

进度条优化：将 dimension-bar 的平滑背景改为 “分段块状” 显示，看起来像命令行里的加载条。

去掉发光（Glow）：移除 .top-glow 和卡片的软阴影，改用简单的实线边框（1px solid）。

动画逻辑：保留数字跳动，但将动画改为瞬时的 “打字机” 风格或快速闪烁，而非柔和的淡入淡出。
